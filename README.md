Project 1 — Static Webpage Design (DecodeLabs Info Site)

A static, semantic webpage built as part of the Frontend Development Internship at DecodeLabs (Batch 2026). This project focuses on mastering the fundamental relationship between HTML structure and CSS styling before introducing any dynamic behavior.

Highlights


Semantic HTML structure (header, main, nav, section, footer)
External CSS only — no inline styles, DRY principles, BEM-style naming
CSS Grid for macro page layout, Flexbox for component alignment
Fully responsive design
Accessible markup with ARIA labels and landmark roles
Single, proper heading hierarchy

Checklist — Project requirements checklist

Footer — Contact details and quote


Tech Stack


HTML5
CSS3 (Grid, Flexbox, custom properties)


Quality Gate


Passes W3C HTML Validator with zero errors
Proper semantic landmarks (nav, main, header, footer)
Pixel-accurate, intentional spacing and typography
